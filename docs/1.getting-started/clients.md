# 🖥️ 客户端白名单模式 {#clients}

:::danger 核心规则：为了服务器安全并防止滥用，琴鸟(Lyrebird) **禁止使用非白名单客户端** 播放。

使用“非白名单”内的客户端登录，您的账号会被系统拒绝登录。如果您确信您当前使用的客户端不存在扫库或者其他损害服务器的行为，请联系腐竹 @窥屏怪 添加UI白名单。
:::
