## 🎬 求片方式说明 {#request-methods}

我们提供“单次求片”和“剧集订阅”两种模式，以满足您不同的需求。

<a id="q-single-request"></a>
### 🧩 自助求片 (种子下载) {#single-request}

:::info 💡 **适用场景**
下载 **电影** 或 **已经更新完毕的整季剧集**。机器人会直接下载您选择的整个种子包。
:::

:::tip ✅ 搜索与筛选小技巧
* **精准筛选**: 搜索结果太多？使用 `/filter` 指令进行内容过滤。例如输入 `/filter S02E4` 即可精准定位第2季第4集。
* **一键切换**: 再次点击 `/filter` 即可取消过滤词，恢复显示全部内容。
* **自动通知**: 选好资源后无需死守，当影片下载完成并入库后，机器人会第一时间为您发送**下载完成**提示。
:::

:::warning ⚡ 下载操作黄金法则 (避免站点冷却)
* **请勿重复搜索!** 一次成功的搜索会提供多个下载指令。请利用一次搜索的结果，找到所有需要的集数指令。
* **操作方式**: 资源底部的下载指令 (例如 `/download_...`) 可以被 **多次点击**。**请勿**每点一集就去再次搜索一次。
* **做种数**: 请务必选择 **做种数 > 10** 或者当前最热门的资源，否则可能导致下载极慢或无法完成。
:::

<a id="q-subscription"></a>
### 🔄 剧集订阅与追更 {#subscription}

:::info 💡 **适用场景**
下载 **正在连载更新的剧集**。订阅后，机器人会自动下载当前已出的所有集数，并在未来更新时自动为您下载。
:::

:::tip ❤️ 收藏即追更
现在您可以更轻松地追剧：只需在机器人或 Emby 中点击 **“我喜欢”/收藏** 您心仪的影片，当该资源后续有新集数更新入库时，机器人会自动为您发送**追更提醒**。
:::

:::warning 订阅须知
* 剧集订阅功能需要 **联系管理员手动订阅**。
* 仅支持订阅 **发布日期在一年内** 的新剧集。
:::

<a id="q-tools"></a>
## 🛠️ 辅助功能：让观影更完美 {#tools}

<a id="q-subtitles"></a>
### 📝 手动上传字幕（限硬盘资源） {#tools-subtitles}

如果下载的影片没有字幕，您可以使用 `/zimu` 命令自行上传。

* **操作方法**: 点击 `/upload_subtitle` 命令，上传 `.srt` 或 `.ass` 格式文件并提供存放路径。
* **命名规范**: 例如 `科洛弗档案 (2008).ASS`。

<a id="q-ai-translate"></a>
### 🤖 AI 智能翻译 {#tools-ai-translate}

对于内封了其他语言字幕的影片，您可以使用 `/translate` 命令进行 AI 翻译。

* **前置条件**: 影片必须是 **`MKV`** 格式，且内部已包含至少一条字幕轨道。
* **效果**: 翻译完成后自动刷新媒体库，您可在字幕选择中看到 AI 翻译的新轨道。

<a id="q-best-practices"></a>
## 👑 求片黄金法则：如何选择最佳资源 {#best-practices}

为了获得最佳的观影体验并避免不必要的问题，请遵循以下建议：

:::danger ❗❗ 严禁下载蓝光原盘 (Important)
**请绝对不要下载蓝光原盘 (Blu-ray / ISO / BDMV) 资源！**
原盘由于其复杂的 **BDMV 目录架构，会导致 EMBY 无法正常识别和播放**。 REMUX 资源是可以正常观看的，但原盘下载后属于“废片”无法观看且极度消耗鸟币。请务必选择兼容性更好的 WEB-DL 或压制版。
:::

:::tip ✅ 优先选择
* **做种数多的资源**: 这是下载成功的首要保证。
* **知名压制组后缀**: 如 `CMCT`, `FRDS`, `ADweb`, `HHweb`, `Mweb` 等，画质与兼容性极佳。
:::

:::warning ⚠️ 谨慎选择
* **HDR 格式**: 如 `Dolby Vision`, `DV`, `HDR10+` 等。若设备不兼容，会导致 **画面严重偏色 (发绿、发紫)**。
:::

:::danger ⛔ 严禁直接下载
* **超大型长篇剧集包**: 例如 `名侦探柯南` (1000+集)、`海贼王` 等整季大包。这会导致索引器崩溃，如有需要请联系管理员。
:::
